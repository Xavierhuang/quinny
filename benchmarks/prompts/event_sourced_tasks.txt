A pure-Python event-sourced task manager. Commands: add-task, complete-task, list-tasks, undo, redo, snapshot, and replay-from-snapshot. All state changes are stored as immutable events in a JSON log; the current state is always derived by folding events. Includes a query engine (find tasks by status, tag, or date range), snapshot compaction (write a snapshot of current state, then replay only events newer than it), and event validation (reject events that would break invariants). Ships a small CLI, self-contained tests, and a demo main.py that exercises the whole flow. Standard library only.
